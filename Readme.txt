To run chefboost_implementation.py:

    pip3 install chefboost
    python3 chefboost_implementation.py

    After running this code a folder called "outputs" will appear. In this folder will be another folder called rules, and in this folder a file called rules.py will
    appear which will contain the created decision tree.
    

To run our_own_implementation.py:
    python3 our_own_implementation.py


    To check all the attribute's gain values, change line 80 print value to "max_attribute_gain"
